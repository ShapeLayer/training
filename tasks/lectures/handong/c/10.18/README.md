# 과제
```
반복하는 계산기 만들기
입력한 만큼 연산을 반복하는 계산기 만들기

-. 반복횟수를 사용자에게 입력 받을 것

-. 더하기, 빼기, 곱하기, 나누기, 모듈러를 반복(+,-,*,/,%)

예시.
loop count에 1을 입력하는 경우 더하기(+)만 계산
loop count에 2를 입력하는 경우 더하기(+), 빼기(-) 계산
loop count에 3을 입력하는 경우 더하기(+), 빼기(-), 곱하기(x)) 계산
loop count에 4를 입력하는 경우 더하기(+), 빼기(-), 곱하기(x), 나누기(/) 계산
loop count에 5를 입력하는 경우 더하기(+), 빼기(-), 곱하기(x), 나누기(/),나머지(%) 계산
loop count에 6을 입력하는 경우 더하기(+), 빼기(-), 곱하기(x), 나누기(/),나머지(%), 더하기(+) 계산

과 같이 반복할 것

- 주의 -
나누기의 경우 소수점 2자리까지 출력
정수만 입력 할 것
loop count에서 예외처리 할 필요 없음(항상 양의 정수만 입력된다고 가정)
0을 입력하는 경우 프로그램 종료
각 연산자별 연산함수를 만들 것
연산자 함수에서는 연산 동작만 할 것 (출력X)
```
